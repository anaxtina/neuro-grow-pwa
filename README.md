# 🌱 NeuroGrow PWA
⚠️ Project Status: Work in Progress (WIP) > This application is currently in active development. New features and neuro-scientific integrations are being added weekly.
A minimalist wellness application designed to harmonize daily productivity with cognitive health. Built with **React** and **Tailwind CSS**, focusing on a high-contrast, distraction-free interface.

## 🧠 The Concept
NeuroGrow is a personal management tool that respects human cognitive limits. By integrating principles of **Neuropsychology** and **User Experience**, the app provides a "dark mode" centered environment to reduce visual fatigue and improve focus.

## 🚀 Technical Highlights
- **PWA (Progressive Web App):** Offline capabilities and installable on mobile devices.
- **Minimalist UI/UX:** Ultra-thin fonts and pure black background for maximum cognitive clarity.
- **Responsive Design:** Optimized for both desktop and mobile performance.
- **Fast Rendering:** Built with Vite for near-instant load times.

## 🛠️ Tech Stack
- **Framework:** React.js
- **Styling:** Tailwind CSS (Dark-theme architecture)
- **Deployment:** Optimized for Vercel/Netlify environments.

## 🔬 Interdisciplinary Approach
This project is part of a larger study on how **Digital Interfaces** affect **Human Behavior**. The choice of typography and color palette is intentionally designed to minimize cognitive load (Cognitive Load Theory).
